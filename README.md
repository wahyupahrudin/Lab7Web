📌 Cara Menjalankan Aplikasi
Opsi 1: Menggunakan Apache (XAMPP) - PALING MUDAH
Langkah 1: Nyalakan Apache & MySQL

Buka XAMPP Control Panel

Klik tombol Start pada Apache

Klik tombol Start pada MySQL

Pastikan keduanya berwarna hijau (Running)

Langkah 2: Akses Aplikasi
Buka browser dan ketik URL:

text
http://localhost/lab11_php_ci/ci4/public/
Opsi 2: Menggunakan Server Bawaan (Spark)
Langkah 1: Buka Terminal

Buka Command Prompt atau terminal di VSCode

Langkah 2: Arahkan ke Folder Project

cmd
cd C:\xampp\htdocs\lab11_php_ci\ci4
Langkah 3: Jalankan Server

cmd
php spark serve
Langkah 4: Akses Aplikasi
Buka browser dan ketik:

text
http://localhost:8080
